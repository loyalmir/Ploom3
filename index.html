사진 분석 (Anthropic API)

사진을 올리면 주변 환경 설명 초안을 자동으로 만들어 줍니다. 키는 이 폰에만 저장되고 어디에도 전송·업로드되지 않습니다.

키 발급: console.anthropic.com → API Keys → Create Key. 처음에 5~10달러만 충전하면 한참 씁니다.
